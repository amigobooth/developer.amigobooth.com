User Authentication Keys API

  • Create a new key
  • Revoke a key’s access

Management of authentication keys via the API requires that you are authenticated.

Create a new key

To create a new key, you must authenticate with HTTP Basic Authentication using your account email address and password.

POST /me/keys

Input

description
Optional string 
{
  "description": "Dwight's iPhone 5"
}

Response

Status: 200 OK
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999
{
  "description": "Dwight's iPhone 5",
  "token": "Mk3W4dTvpdUNar7AB3RD4PBoT8Py9wFNZ7WakvQP",
  "created_at": "2012-12-09T17:58:26Z"
}

Revoke a key’s access

DELETE /me/keys/:token

Response

Status: 204 No Content
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999
